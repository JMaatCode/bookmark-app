# 预备

1. 安装nvm，管理node版本：https://github.com/nvm-sh/nvm | https://github.com/coreybutler/nvm-windows
2. 安装node，版本20：nvm install 20.18.0
3. 切换node版本：nvm use 20.18.0

# 运行

1. npm i
2. npm run dev
